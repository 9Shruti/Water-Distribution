# E-Commerce Water Distribution

A full-stack PHP & MySQL web application for water distribution.

## Features
- User Registration & Login
- Product Catalog
- Shopping Cart
- Order Management
- Admin Dashboard
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to Run
1. Install XAMPP.
2. Copy the project to htdocs.
3. Import the SQL file into phpMyAdmin.
4. Start Apache and MySQL.
5. Open http://localhost/E-Commerce-Water-Distribution/
